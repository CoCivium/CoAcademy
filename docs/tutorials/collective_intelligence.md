# 🐾 Collective Intelligence

Collective intelligence happens when multiple minds become more useful together than apart.

That can include:
- humans
- AIs
- teams
- communities
- odd little future swarms

## ✨ Why you should care
Because one lonely genius is fragile.  A good network can keep learning even when one node gets tired, lost, or dramatic.

## 🧩 CoScrollSnack
Think of intelligence less like a lone wizard and more like a jam session with receipts.
